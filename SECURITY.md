# Security Policy

Hadirin menyimpan data secara lokal di perangkat pengguna. Tidak ada backend bawaan dan aplikasi
tidak mengirim data siswa ke server proyek.

## Melaporkan Masalah

Jika menemukan celah keamanan, jangan buka issue publik berisi data sensitif. Kirim laporan pribadi ke
maintainer repo terlebih dahulu, lalu maintainer akan menyiapkan perbaikan dan publikasi yang aman.

## Data Siswa

Pengguna bertanggung jawab menjaga perangkat, backup, dan file export Excel. Hindari membagikan backup
atau export yang berisi data siswa ke pihak yang tidak berwenang.
