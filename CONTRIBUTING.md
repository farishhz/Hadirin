# Contributing

Terima kasih sudah tertarik membantu Hadirin.

## Prinsip Produk

- Tetap offline-first.
- Jangan menambah backend wajib untuk fitur inti.
- Prioritaskan alur operator sekolah yang non-teknis.
- Gunakan istilah Indonesia yang familiar.
- Jaga scope kecil: absensi, rekap, import, export.

## Cara Kontribusi

1. Fork repo.
2. Buat branch fitur atau perbaikan.
3. Jalankan test dan build sebelum pull request.
4. Jelaskan masalah yang diselesaikan dan cara mengeceknya.

```bash
npm install
npm test
npm run build
```

## Area Kontribusi yang Dibutuhkan

- Template Excel untuk berbagai format sekolah.
- Perbaikan aksesibilitas.
- Terjemahan istilah ke bahasa daerah atau bahasa Inggris.
- Format export rekap yang lebih cocok untuk sekolah tertentu.
- Dokumentasi penggunaan untuk operator non-teknis.
